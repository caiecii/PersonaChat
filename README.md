# PersonaChat

PersonaChat is a fully local Android app, which is BYOK, and stores all your data on your phone and your phone only. No accounts, no limits, no subscriptions, no drama.


## Features

- Chat labelling (to label your roleplays)
- Creature comforts from Cai (chat colour changes, rewinds, swipes, message editing, pinned messages, archived chats)
- App UI colour options
- App/Chat UI font options
- Delete your chats fully. No bringing back ancient history if you don't want.
- Custom instructions. Like lorebook, but it actually exists
- Import and Export data - so you don't lose anything if you transfer devices.
- Unlimited Characters/Personas
- No subscription model
- Uses Gemini API

## Frequently Asked Questions

**What does BYOK mean?**
'Bring Your Own Key'. PersonaChat uses Gemini API keys to function. To find out how to get your own API key, check here > https://ai.google.dev/gemini-api/docs/api-key

**Why can't I send a message?**
Either you haven't configured your API key correctly, or you're being rate limited. If you're not a paying user (i.e. you haven't bought Gemini credits, which is totally optional), I'd recommend using Gemma 4 31B since it has unlimited messages - the other models are there for power users.

**Why is there no feed?**
Because everything is local, and I mean everything. The only characters are the ones you build, so you can totally curate your experience and only interact with what you want.

**Why not make it more like Character AI?**
Because online functionality requires servers and servers cost money. No one here wants another subscription model.

**My Character/Persona's image disappeared!**
Images are stored locally so if you move/delete the image you added to it, it won't appear anymore. I'd recommend making a folder on your phone to store these.

**Will you ever support iOS/iPadOS?**
No. That costs money, and requires a Mac, both things I don't have access to.
